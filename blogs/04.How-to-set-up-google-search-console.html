Setting Up Google Search Console With Your Site

Google Search Console + SEO + Analytics + Site Verification

Published: October 2023

Back to Blog

Setting up Google Search Console (formerly Google Webmaster Tools) is a free and essential step for anyone who wants to optimize their website's SEO. This guide will walk you through the entire setup process for your GitHub-hosted portfolio.

Step 1: Add Your Site

  • Go to Google Search Console and add your website using the URL Prefix option
  • Click Continue to proceed to verification

Step 2: Verify Ownership

  • Download the provided .html verification file
  • Place the file in your site's root directory. If you're using Visual Studio Code, drop it directly into your repo folder

Step 3: Upload to GitHub

Open your terminal and run:

git add googleea4877ce6dc12412.html
git commit -m "Add Google Search Console verification"
git push origin main

Step 4: Complete Verification

  • Back in Google Search Console, click Verify

Visual Walkthrough

Here's what the process looks like step by step:

After uploading the file to GitHub, you should see it in your repository. Then click verify in Google Search Console.

What You'll Get

Once verified, you'll have access to valuable SEO data including search performance, indexing status, mobile usability, and more. Google Search Console is essential for monitoring your website's presence in Google search results.
