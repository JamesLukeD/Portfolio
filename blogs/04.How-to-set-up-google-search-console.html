How to set up Google Search Console with GitHub

In this blog I will show you how simple it is to add Google Search Console!

  • Add Your Site

    • Go to Google Search Console and add your website using the **URL Prefix** option.
    • Click Continue to proceed to verification.
  • Verify Ownership
    • Download the provided `.html` verification file.
    • Place the file in your site’s root directory. If you're using Visual Studio Code, drop it directly into your repo folder.
  • Upload to GitHub
    • Open your terminal and run:
    • `git add googleea4877ce6dc12412.html git commit -m "Add Google Search Console verification" git push origin main`
  • Complete Verification
    • Back in Google Search Console, click Verify.
    • If it doesn’t work immediately, wait a moment and try again. GitHub can take a few seconds to update.

Practical View

First add you websites URL into the PreFix and click 'select' once you've gone through this stage you will be directed into a verification area. Here you will be required to download the html file and upload it onto your root directory of the website.

As you can see here I've just dropped the file into my root directory inside my visual studio code.

I then used the commands


        git add 'googleea4877ce6dc12412.html'
        git commit -m "Google Search Console"
        git push origin main

To upload this file directly into my GitHub file where the portfolio lives. Open your GitHub to verify that the file has correctly uploaded and then click verify. You may need to wait a few moments for GitHub to catch up with the upload by retrying the verification a few times.

There you have it, you should now be verified as the website owner and have access to the performance and other options linked to your website With Google Search Console!
