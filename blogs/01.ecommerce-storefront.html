How I Built a Full-Stack E-commerce Storefront with React and Node.js

React + Node.js + Express + JSON API

Published: January 2024

Back to Blog

In this project, I set out to build a fully responsive e-commerce storefront using a full-stack approach ‚Äî React on the front-end and Node.js with Express on the back-end. My goal was to turn a static site into a dynamic shopping experience with product listings, cart logic, and an API-based data source.

Starting with Static HTML/CSS/JS

I began by designing the structure using 5 main sections:

  • Banner
  • Header
  • Hero Section
  • Product Grid
  • Footer

Rebuilding as a React App

I split the static components into reusable React components ‚Äî like Banner.jsx, Navbar.jsx, and ProductList.jsx ‚Äî and styled them via a global stylesheet.

Introducing Backend with Node + Express

I created a lightweight Express server to deliver product data via a JSON file:

  • Set up products.json
  • Created a /api/products route
  • Enabled CORS so React could fetch from localhost

Replacing Static with Dynamic Data

On the frontend, I replaced the static array with a useEffect() hook to fetch live data from the backend.

What I Learned

  • React component structure and state management
  • Building a simple REST API with Express
  • Using CORS and fetch to connect client and server
  • Planning for scalability with JSON data (future MongoDB)

Next Steps
